# Central Bistro — Nav Page

**Навигационная страница** для ресторана Central Bistro.  
Доступна по адресу: [go.bistro-central.ru](https://go.bistro-central.ru)  
Разработана для перехода через QR-код.

## Стек
- React + Vite
- styled-components
- framer-motion
- lucide-react

## Запуск

```bash
npm install
npm run dev
```

## Деплой

Проект деплоится на Vercel. Домен `go.bistro-central.ru` подключён через CNAME.

```bash
npx vercel --prod --yes
```
